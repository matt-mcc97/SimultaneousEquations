# SimultaneousEquations
Generates simultaneous Equations of different varieties, for use in creating worksheets and examples in Secondary Mathematics lessons.

The three types of simultaneous equations it will generate are 

1. Equations with the same term in both
2. Equations where one must be multiplied to find same term
3. Equations where both equations must be multiplied to find the same term

As well as these three types of equations, it will also give the option of whether or not you want negative answers or coefficients, as well as decimal answers. So far, the decimal answers are only .5s.

The coefficients will generate randomly from -10 to 10, and it will detect when equations are unsolveable and not include them.

The questions are generated by first randomly generating the answers, then generating coefficients that match the conditions, then once I have these, the answers to each equation are generated.
